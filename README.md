# Agiliza-TI

AgilizaTI é uma plataforma de suporte técnico desenvolvida como proposta acadêmica, utilizando ASP.NET, conforme solicitado pela faculdade. O objetivo do sistema é trazer organização e agilidade para o ambiente de TI, permitindo o registro de chamados, acompanhamento de status e gerenciamento das demandas de forma clara e centralizada.
A plataforma foi pensada para facilitar tanto o trabalho da equipe técnica quanto a experiência do usuário, oferecendo uma estrutura simples, funcional e intuitiva.
