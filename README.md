# Agiliza-TI

AgilizaTI é uma plataforma de suporte técnico desenvolvida como proposta acadêmica, com foco em organizar atendimentos, registrar chamados e tornar o fluxo de soluções mais ágil dentro da área de TI.
A ideia é oferecer um sistema simples, funcional e eficiente, permitindo que usuários e equipes acompanhem problemas, status e resoluções de forma clara, mantendo o suporte sempre organizado e acessível.
